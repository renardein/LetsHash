# Let's Hash!
Let's Hash is a simple C# based tool for get string hash.
# Functions

  - Get BASE64 from string
  - Get string from BASE64
  - Get MD5 from string
  - Get SHA1 from string
  - Get SHA256 from string
  - Get SHA512 from string
  - Get SHA384 from string
